# Credit_Rating_Analysis_Project

## 신용등급 확인 프로젝트 소개

해당 프로젝트는 신용등급 확인을 할 수 있게 만드는 프로젝트입니다.

해당 프로젝트에서 Hadoop Eco-system을 활용하여 데이터 수집부터 시작하여 데이터 저장 및 처리, 분석까지 진행하였으며 웹에서 데이터를 입력하면 다중회귀분석을 통하여 데이터에 따른 신용등급을 표시해주는 기능까지 구현하였습니다.
