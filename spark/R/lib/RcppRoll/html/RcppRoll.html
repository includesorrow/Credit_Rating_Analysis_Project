RcppRoll

Description

This package implements a number of 'roll'-ing functions for R vectors and matrices.

Details

Currently, the exported functions are:

  • roll_max

  • roll_mean

  • roll_median

  • roll_min

  • roll_prod

  • roll_sd

  • roll_sum

  • roll_var
